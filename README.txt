# Phase 5 Documentation

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Sawiresz/Childcare_PE_Analysis.git
   ```

2. Install necessary dependencies:
   ```bash
   cd Childcare_PE_Analysis
   npm install
   ```

3. Set up your environment variables:
   - Create a `.env` file in the root directory and add the required variables.

4. Run the application:
   ```bash
   npm start
   ```

## Financial Statement Definitions
- **Assets**: Resources owned by the childcare centres that are expected to provide future economic benefits.
- **Liabilities**: Obligations the centres owe to external parties.
- **Equity**: The residual interest in the assets of the childcare centres after deducting liabilities.
